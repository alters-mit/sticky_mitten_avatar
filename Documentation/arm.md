# Arm

`from sticky_mitten_avatar import Arm`Arm`

An enum that defines the side that an arm is on.

```python
from sticky_mitten_avatar import Arm

for arm in Arm:
    print(arm) # Arm.left, Arm.right
```

| Value | Description |
| --- | --- |
| `left` | The left arm. |
| `right` | The right arm. |
